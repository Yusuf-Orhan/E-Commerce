# E Commerce
## Project Features
 - MVVM Architecture
 - Coroutines
 - Navigation Components
 - Hilt for Dependency Injection
 - Retrofit
 - Firebase Auth & Firestore
 - Room Database
 - Glide
 - TabLayout & ViewPager2 with PageTransformer
 - Lottie Animation
